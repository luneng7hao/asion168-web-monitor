## [1.0.0] - 2024-12-05

### 新增
- ✨ 完整的错误监控功能（JavaScript 错误、资源加载错误、Promise 错误）
- ✨ 性能监控（页面加载时间、Web Vitals）
- ✨ 用户行为追踪（PV/UV、点击、路由变化）
- ✨ 接口监控（API 请求成功率、响应时间）
- ✨ 实时 Dashboard 数据可视化
- ✨ 基于 Elasticsearch 的日志查询功能
- ✨ 多平台 SDK 支持（Web、微信小程序、React、Vue2/3、Svelte）
- ✨ MidwayJS 后端实现
- ✨ Spring Boot 后端实现
- ✨ Vue3 管理端
- ✨ Docker 一键部署支持
- ✨ 数据清理功能

### 技术栈
- 后端：MidwayJS (Node.js) / Spring Boot (Java)
- 数据库：MongoDB + InfluxDB + Redis + Elasticsearch
- 前端：Vue 3 + TypeScript + Vite + Element Plus
- SDK：TypeScript，支持多框架
